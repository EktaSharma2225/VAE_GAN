# Deep Learning Projects: VAE & GAN

This repository contains implementations of two popular generative deep learning models built using **PyTorch**.

## Projects

### Variational Autoencoder (VAE)
- Encoder–Decoder architecture
- Reparameterization trick
- Latent space learning
- Reconstruction using Reconstruction Loss + KL Divergence

### Generative Adversarial Network (GAN)
- Generator and Discriminator networks
- Adversarial training process
- Synthetic image generation
- Training progress visualization

## Tech Stack
- Python
- PyTorch
- NumPy
- Matplotlib

## Learning Outcomes
- Understanding latent space representation
- Implementing generative deep learning models
- Training and evaluating VAE and GAN architectures
- Visualizing generated and reconstructed images
